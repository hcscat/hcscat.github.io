## 문제 환경

공공 업무 시스템은 화면 개발만으로 끝나지 않고 UI 표준, 개발 가이드, ERD, 인터페이스 문서, 단위테스트 산출물을 함께 맞춰야 합니다. 이 프로젝트에서는 화면 동작과 백엔드 CRUD를 함께 보면서 업무 흐름을 구현했습니다.

## 수행한 일

- exBuilder6 편집 도구와 JavaScript를 사용해 업무 화면 동작을 구현했습니다.
- Java, Spring, MyBatis 기반으로 Oracle DB에 대응하는 등록, 수정, 조회 로직을 구현했습니다.
- 내부 메타정보, 표준단어, 도메인, 논리/물리 DB 구조를 참고해 설계 산출물 작성을 지원했습니다.
- PDF 첨부파일 미리보기 기능에서 CORS, 바이너리 스트림, Blob, pdf.js 렌더링 흐름을 다뤘습니다.

## 포트폴리오에서 강조할 점

이 경험은 최신 프론트엔드 기술보다 업무 시스템 개발에서 중요한 문서 기반 분석, 화면과 백엔드 연결, DB 흐름 이해, 산출물 대응 역량을 보여주는 사례입니다.
