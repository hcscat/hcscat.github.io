## 프로젝트 개요

공공 업무 시스템 구축 프로젝트에서 업무 화면 개발과 데이터 처리 흐름 구현을 담당했습니다. Eclipse IDE와 exBuilder6 플러그인을 사용해 화면을 개발했고, 화면 이벤트는 JavaScript 기반으로 구현했습니다.

## 수행한 일

- exBuilder6 기반 업무 화면을 개발했습니다.
- JavaScript로 화면 이벤트와 입력값 처리 흐름을 구현했습니다.
- XML SQL 파일을 작성해 조회, 등록, 수정, 삭제 흐름을 처리했습니다.
- 개발가이드와 화면 흐름을 기준으로 구현 내용과 산출물을 정리했습니다.

## 업무 강점

공공 업무 시스템은 화면 구조, 입력 조건, 조회 결과, 저장 흐름을 문서와 함께 맞춰야 합니다. 이 프로젝트에서는 화면 개발과 데이터 처리 흐름을 함께 보면서 업무 시스템 개발 경험을 확장했습니다.
