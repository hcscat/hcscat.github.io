## 문제 환경

사회보장정보원 입양업무관리시스템은 공공 업무 영역의 내부 시스템입니다. 일부 외부 서비스 URL은 존재하지만, 실제 담당 화면과 내부망 화면은 공개하기 어렵습니다.

## 수행한 일

- Eclipse IDE에서 exBuilder6 플러그인을 사용해 업무 화면을 개발했습니다.
- 화면 동작은 JavaScript 기반으로 구현했습니다.
- 백엔드 로직을 크게 직접 작성하기보다는 exBuilder와 연동되는 XML SQL 파일을 작성해 CRUD 흐름을 처리했습니다.
- 개발가이드와 화면 흐름을 기준으로 입력, 조회, 저장 동작을 구현했습니다.

## 포트폴리오에서 강조할 점

이 경험은 최신 기술 쇼케이스보다 업무 시스템의 화면 구조, 입력/조회/저장 흐름, 공공 프로젝트의 공개 제약을 이해하고 작업한 사례로 정리합니다.
