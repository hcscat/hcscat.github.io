## 현재성

최근 경력은 SAP UI5/Fiori 기반 기업 솔루션 화면 운영개발입니다. 포트폴리오에서는 "최신 프론트엔드 프레임워크 전문가"가 아니라 "기업 업무 화면을 분석하고 안정적으로 수정하는 운영개발자"로 설명합니다.

## 수행한 일

- SAP UI5/Fiori 화면 구조와 JavaScript 로직을 분석했습니다.
- CSS 수정, 화면 배치, 컴포넌트 표현 조정 등 운영개발성 작업을 수행했습니다.
- 고객사 요청사항을 바탕으로 기존 화면의 변경 범위와 영향도를 확인했습니다.
- Git 기반으로 변경 사항을 관리했습니다.

## 공개 범위

회사 공개 정보상 PSNC는 SAP HCM, SAPUI5, SuccessFactors 영역의 컨설팅과 개발을 주요 사업으로 소개합니다. 본 포트폴리오에서는 이 공개 범위와 본인의 화면 운영개발 역할만 연결합니다.

SAP 백엔드와 OData/API 연계는 백엔드 담당 범위와 연결되므로, 직접 담당 업무처럼 과장하지 않습니다.
