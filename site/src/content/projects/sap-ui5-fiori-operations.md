## 프로젝트 개요

기업 솔루션 환경에서 SAP UI5/Fiori 기반 업무 화면을 운영개발하고 있습니다. 기존 화면 구조를 분석하고, 요청 사항에 맞춰 JavaScript 로직과 CSS 표현을 수정합니다.

## 수행한 일

- SAP UI5/Fiori 화면 구조와 JavaScript 로직을 분석했습니다.
- 화면 배치, 컴포넌트 표현, CSS 스타일을 조정했습니다.
- 운영 중인 화면의 변경 범위와 영향도를 확인했습니다.
- Git 기반으로 변경 사항을 관리했습니다.

## 업무 강점

업무 화면은 단순한 UI 수정만으로 끝나지 않기 때문에 기존 동작, 데이터 흐름, 사용자 요청을 함께 확인해야 합니다. 이 경험을 통해 운영 중인 기업 솔루션 화면을 안정적으로 분석하고 수정하는 역량을 쌓고 있습니다.
