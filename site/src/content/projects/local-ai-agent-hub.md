## 문제의식

반복적으로 확인하는 채용 정보, 기술 뉴스, 관심 종목, 제품 아이디어를 매번 수동으로 찾아보면 맥락이 끊깁니다. 이를 로컬 서버가 주기적으로 수집하고 웹 대시보드와 알림으로 보여주는 개인 운영 도구로 만들었습니다.

## 구성

- FastAPI 서버로 대시보드, 수동 질의, 즉시 실행 API를 제공합니다.
- APScheduler로 수집, 아이디어 생성, 요약 전송, 컨텍스트 정리를 주기 실행합니다.
- SQLite에 수집 결과, 아이디어, 실행 이력, 질의 응답을 저장합니다.
- API 키와 웹훅은 `.env`로 분리해 공개 문서에 노출하지 않습니다.

## 포트폴리오에서 강조할 점

이 프로젝트는 "AI가 모든 것을 대신한다"가 아니라, 관심 정보를 수집하고 실행 이력을 남기며 다음 작업의 맥락으로 재사용하는 작은 운영 시스템을 설계한 사례입니다.
