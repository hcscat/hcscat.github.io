## 프로젝트 개요

반복적으로 확인하는 정보와 작업 아이디어를 로컬 서버에서 수집하고 정리하는 개인 자동화 도구를 만들었습니다. 단발성 스크립트가 아니라 대시보드, 스케줄러, 저장소, 알림 채널을 갖춘 작은 운영 시스템으로 구성했습니다.

## 구성

- FastAPI 서버로 대시보드, 수동 질의, 즉시 실행 API를 제공합니다.
- APScheduler로 수집, 요약, 실행 이력 정리를 주기 실행합니다.
- SQLite에 수집 결과, 아이디어, 실행 이력, 질의 응답을 저장합니다.
- API 키와 웹훅은 환경 변수로 분리해 관리합니다.

## 구현 포인트

이 프로젝트는 정보를 한 번 조회하는 데서 끝내지 않고 실행 이력과 결과를 저장해 다음 작업의 맥락으로 다시 사용할 수 있게 만든 사례입니다. 운영 도구처럼 실행 상태를 확인하고, 필요한 시점에 수동 실행할 수 있도록 구성했습니다.
