기술 경력은 기술명만 나열하는 것보다 직무, 도메인, 운영 방식, 사용 기술을 함께 정리해야 읽기 쉽습니다.

## 정리 기준

- Java Spring 업무 시스템 개발
- Java Spring 운영개발
- Java MyBatis 공공 시스템
- Java JSP Nexacro
- exBuilder Java Oracle
- SAP UI5 Fiori JavaScript
- 이커머스 운영개발
- 백오피스/관리자 화면 개발

## 담당 범위와 구분하는 영역

- React/TypeScript 단독 프론트엔드보다는 업무 시스템 운영개발 경험을 중심으로 설명합니다.
- Node.js 단독 백엔드보다 Java/Spring/MyBatis 기반 경험을 중심으로 설명합니다.
- DevOps/SRE나 모바일 앱 전담 업무보다 웹 업무 시스템과 백오피스 경험을 중심으로 설명합니다.

## 구조화한 이유

같은 경력이라도 기술, 도메인, 운영 방식 중 어느 축을 먼저 보여주느냐에 따라 전달력이 달라집니다. 이 기록은 Java/Spring 업무 시스템 개발자라는 방향 안에서 경력을 일관되게 설명하기 위한 정리입니다.
